API接口案例

基于MySQL数据库 + Express对外提供用户列表的API接口服务

  1. 技术栈：
    • 第三方包express、mysql2
    • ES6模块化
    • Promise
    • async/await
  2. 主要实现步骤
    1. 搭建项目的基本结构
    2. 创建基本的服务器
    3. 创建db数据库操作模块
    4. 创建user_ctrl 业务模块
    5. 创建user_router路由模块

实现

  1. 初始化包管理配置文件package.json —— npm init -y

  2. 启用ES6模块化支持

    • 在package.json中声明"type": "module"

  3. 安装第三方依赖包

    • 运行npm install express@4.17.1 mysql2@2.2.5

  4. 创建最基本的web服务器

  5. 创建db数据库操作模块

  6. 创建user_ctrl 业务模块

  7. 创建user_router路由模块
